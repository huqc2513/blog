# express简易实现

express是一个简单、灵活的node框架,使用市场占有率达56%。

# 实现原理

express是在http模块的上层分装，通过http模块的createServer函数，将app函数做为监听器，且每个请求过来之后，采用数组保存需要调用的请求路径回调，

# 实现功能点

  • get,post,put,all等方法
  • 中间件
  • 路径参数解析

# app函数以及listen实现

从测试用例上分析，express是一个函数，它返回另新一个函数app，app函数是一个监听器，挂载着各种方法

http模块中的createServer函数，可以传入一个函数app，客户端所有请求过来都会调用这个监听器

  //test.js
  const express = require('./express')
  const app = express()
  app.get('user',function(req,res){
    res.end('user')
  })
  app.listen(4001, () => console.log('app listening on port 4001!'))

//exporess.js
function createExpress (){

    }
    app.listen = function(){
      let server =  http.createServer(app);
      server.listen.apply(server,arguments);
    }
}

module.exports = createExpress
